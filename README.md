# Shopping List

The project implements a Shopping List App for Mobile devices. 

Click here for a demo video: [https://github.com/adwiteeya-r-paul/ShoppingList/issues/1]

## Project Details:

- The project was built on Android Studio in Kotlin. 
- It uses the Navigation Backstack, Dialog Boxes, the Room Database with its basic SQL queries and DAO functionalities.
- It supports:
  1. a filter functionality to sort the items based on prices
  2. Sound Effects during transitions and
  3. different icons for each food category.
  4. Editing and deleting items added previously and checking off items already bought.
