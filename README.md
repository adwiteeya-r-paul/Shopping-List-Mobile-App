# Shopping List

The project implements a Shopping List App for Mobile devices. Click here for a demo video: [https://github.com/adwiteeya-r-paul/ShoppingList/issues/1]

## Project Details:

The project was built on Android Studio. It uses the Navigation Backstack, Dialog Boxes, the Room Database with its basic SQL queries and DAO functionalities. The language used is Kotlin. It supports a filter functionality to sort the food based on prices, also uses Sound Effects during transitions and assigns different icons for each food category.
